# 土量配分計画ナビ Ver.1.0

掘削・埋戻し数量から流用土・残土・購入土・仮置きを計算し、土量配分図とExcel帳票で施工前の判断を支援する建設現場向けWebアプリです。

## 主な機能

- 流用土・残土・購入土・仮置き量の計算
- 土質別・構造物別の集計
- 土量配分図
- 検算と入力警告
- JSON保存・読込
- Excel帳票出力

## 利用上の注意

本ツールは施工計画の検討を支援する補助ツールです。
正式な設計、契約数量、施工条件については、関係資料や発注者との協議内容を確認してください。

本リポジトリにはオープンソースライセンスを付与していません。
